# 05 — Estructura de Chats XX-MAIN / XX-EXERCISES / XX-WORK

Estos chats definen cómo interactúa el DoJo con la IA (ChatGPT) para enseñar, practicar y resolver.

---

# 1. XX-MAIN — Teoría mínima + ejemplos + mini-misión

Rol de la IA:
- Explicar teoría **solo la necesaria**  
- Dar ejemplos concretos  
- Conectar todo con la misión activa  
- Mantener el estilo MAIN-BASE (calma + comprensión)  

Estructura dentro del chat:
1. Concepto  
2. Ejemplo  
3. Mini-ejercicio  
4. Conexión con misión  
5. Reflexión breve  

---

# 2. XX-EXERCISES — Generador de misiones

Rol de la IA:
- Crear misiones pequeñas  
- Variar niveles de dificultad  
- Diseñar side missions  
- Proponer boss missions cuando toque  

Estructura:
1. Lista de misiones principales  
2. Side missions  
3. Mini-retos  
4. Misión final (si aplica)  

---

# 3. XX-WORK — Acompañamiento técnico

Rol de la IA:
- Debugging  
- Explicación paso a paso  
- Refactor  
- Revisión de código  
- Pensar en voz alta contigo  
- Acompañar bloqueos  

Es un “tutor técnico” dentro del DoJo.

---

# Flujo de estudio con los chats

1. Vas a **EXERCISES** → eliges misión.  
2. Vas a **MAIN** → recibes teoría mínima necesaria.  
3. Vas a **WORK** → resuelves errores o bloqueos.  

---

# Objetivo de esta estructura

- Orden  
- Coherencia  
- Flujo constante  
- Evitar dispersión  
- Separar roles claramente  
- Mantener claridad del DoJo  

---

Este documento formaliza la interacción entre el sistema DoJo Study y la IA.
