## Objective
Utilizar la librería estándar `datetime` para obtener y formatear la fecha actual.

---

## Description
El uso de librerías estándar es esencial. Esta misión introduce la importación de módulos y el uso de funciones de tiempo para generar marcas temporales (timestamps).

Conceptos clave:
- `import`.
- Objeto `datetime`.

---

## Required Knowledge
- `import datetime`
- `datetime.datetime.now()` o `date.today()`

---

## Steps
1. Crear un script de prueba o modificar uno existente.
2. Importar el módulo `datetime`.
3. Obtener la fecha y hora actual.
4. Imprimir la fecha con un formato legible (ej. YYYY-MM-DD).

---

- [ ] Se utiliza el módulo `datetime`.
- [ ] Se imprime la fecha actual del sistema.
