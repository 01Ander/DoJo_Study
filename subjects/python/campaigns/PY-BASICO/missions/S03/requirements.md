## Objective
Ejecutar código Python de forma interactiva sin crear archivos.

---

## Description
El REPL (Read-Eval-Print Loop) permite probar fragmentos de código rápidamente. Esta misión consiste en familiarizarse con el entorno interactivo.

Conceptos clave:
- Intérprete interactivo.
- Ejecución inmediata.

---

## Required Knowledge
- Comando `python` (sin argumentos).
- Salir del REPL (`exit()` o `Ctrl+D`).

---

## Steps
1. Abrir la terminal.
2. Iniciar el intérprete de Python.
3. Ejecutar operaciones matemáticas básicas.
4. Manipular strings directamente.
5. Asignar variables y operarlas.
6. Salir del intérprete.

---

## Completion Criteria
- [ ] Se ejecutan comandos directamente en el shell de Python.
- [ ] Se comprende la diferencia entre modo script y modo interactivo.
